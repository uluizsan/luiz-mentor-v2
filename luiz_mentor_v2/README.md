# Luiz Mentor - Protocolo Arquiteto (Web App)

Aplicativo web mobile-first focado em produtividade e disciplina estratégica. Construído com Vanilla JS, HTML e CSS Premium.

## Estrutura do Projeto

- **index.html**: Ponto de entrada da aplicação.
- **css/**: Estilos globais e tokens de design (Dark Theme).
- **js/**:
  - **app.js**: Controlador principal e rotas.
  - **components/**: Widgets da interface (Dashboard, Tarefas, Pânico, Gravação).
  - **services/**: Lógica de simulação de Backend e Rotina.

## Como Rodar Localmente

Basta abrir o arquivo `index.html` em qualquer navegador moderno. Recomendado usar o modo de visualização móvel (F12) para simular o layout do Motorola G15.

## Como Fazer Deploy na Vercel

1. Faça upload deste projeto (ou conecte seu repositório Git).
2. A Vercel detectará automaticamente como um site estático.
3. Não é necessário comando de build ou configurações extras.
4. Clique em **Deploy**.

## Tecnologias

- HTML5
- CSS3 (Variables)
- JavaScript (ES Modules)
- Ícones: Lucide (via CDN)
- Fontes: Google Fonts (Inter & Outfit)
